  • What is My-blogger
    It is normal blogging web application to have Q&A or disussion forum, similar to twitter, wordpress, etc
  • How to Use
    As you login you will have Main Discussion Topic, Each Main Topic willhave many sub topic for discussion to start.
    Sub Topic has Drop down Menu button for view, rise new discussion, subscribe and unsusbscribe. Any discussion or
    comments added will be mailed to registered mail id's of subscriber. Mail will have hyper link to respond the comment without loggin to Blogger.
  • Technical
    Perl, different Perl modules(CGI,DBI,MAIL,Templet, etc), CSS, Bootstrap UI architecture, HTML, Ajax, Jquery, Javascript, MySQL.
